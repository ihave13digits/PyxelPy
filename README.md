A pixel art editor written in Python, with Pygame.

Required Libraries:

	Pygame

Key Commands:

	Esc		    -       Exit
	
	Ctrl+s		-       Quick Save
	
	Ctrl+v		-       Paste Into Canvas
	Shift+v		-       Paste Into Selection
	
	Ctrl+g		-       Remove Grid
	Shift+g		-       Apply Grid
	
	Arrow Keys  -	    Move Cursor
